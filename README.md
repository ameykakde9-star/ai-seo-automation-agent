# AI SEO Automation Agent

## 🚀 Overview
This project automates SEO content generation using no-code tools (n8n) and free APIs.

## ⚙️ Workflow Steps
1. Fetch search results using SerpAPI
2. Extract top 5 results using Code node
3. Generate SEO-optimized content using Groq API
4. Store results automatically in Google Sheets

## 🧠 Features
- Keyword-based search automation
- AI-generated SEO blog content
- Real-time data processing
- Automated reporting system

## 🛠 Tools Used
- n8n
- SerpAPI
- Groq API
- Google Sheets

## 📊 Output
Each run generates:
- Title
- Link
- SEO Content
